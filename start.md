# Getting Started (Zero to Hero)

Follow these instructions to run the project on a completely new PC.

## 1. Install Prerequisites

1.  **Node.js**: Download and install the LTS version from [nodejs.org](https://nodejs.org/).
2.  **Git**: Download and install from [git-scm.com](https://git-scm.com/).
3.  **Expo Go App**: Install the **Expo Go** app on your iOS or Android phone from the App Store or Play Store.

## 2. Setup Project

1.  Open a terminal (Command Prompt, PowerShell, or Terminal).
2.  Navigate to the project folder:
    ```bash
    cd path/to/fintrackorApp
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

## 3. Run the App

1.  Start the development server:
    ```bash
    npx expo start
    ```
2.  **On your phone**: Open **Expo Go** and scan the QR code displayed in the terminal.
    - **Android**: Use the Expo Go app to scan.
    - **iOS**: Use the default Camera app to scan.

## Optional: Run on Emulator/Simulator

- **Android**: Install [Android Studio](https://developer.android.com/studio), set up a virtual device, and press `a` in the terminal after running the start command.
- **iOS (Mac only)**: Install [Xcode](https://developer.apple.com/xcode/), and press `i` in the terminal after running the start command.
